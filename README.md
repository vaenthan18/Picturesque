# Picturesque: A Lightweight Photo Editor
Experimentation with Buffered Image manipulation in JAVA through applying it to a watered down photo editor.

Features Include:

+ Image flips
+ Grayscale
+ Sepia Tone
+ Invert Colours
+ Gaussian Blur
+ Bulge
+ Rotate
+ Scale
+ Saving
+ Undoing and Redoing
+ Restoring the Original Image
